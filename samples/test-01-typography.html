Начертание (font-weight / font-style)

Обычный — Жирный (bold) — Тонкий (300) — Курсив (italic) — Жирный курсив

Декорации текста (text-decoration-line)

Без декорации — Подчёркнутый (underline) — Зачёркнутый (line-through) — Надчёркнутый (overline)

Цвет текста (color) — hex #rrggbb

Красный #e53e3e — Синий #3182ce — Зелёный #38a169 — Фиолетовый — Оранжевый — Бирюзовый — Золотой

Трансформации текста (text-transform)

текст uppercase — ВЕРХНИЙ РЕГИСТР — ТЕКСТ lowercase — нижний регистр — первая буква каждого слова — capitalize

Межбуквенный интервал (letter-spacing)

Тесно -1px — Норма 0px — Широко +3px — Очень +7px

Межсловный интервал (word-spacing)

Слова тесно -4px — Слова нормально 0px — Слова широко +10px

Выравнивание текста (text-align)

← text-align: left — по левому краю (по умолчанию для LTR)

↔ text-align: center — по центру ↔

text-align: right — по правому краю →

Межстрочный интервал (line-height) — 2 строки в каждом

line-height: 1.0 — плотный. Первая строка текста нейронной сети. Вторая строка прижата к первой — интервал минимален.

line-height: 2.0 — разреженный. Первая строка с большим воздухом. Вторая строка далеко от первой — двойной интервал.

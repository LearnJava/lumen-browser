AVIF Format Information

AVIF (AV1 Image File Format) is a modern image format based on the AV1 video codec.

  • Container: ISOBMFF (ISO Base Media File Format)
  • Codec: AV1 (Alliance for Open Media)
  • Advantages: Better compression than JPEG/WebP, supports HDR
  • Browser Support: Modern browsers (Chrome 85+, Firefox 93+, Safari 16+)

display: block — каждый блок начинается с новой строки, занимает всю ширину
Блок 1 (div) — display: block — вся ширина
Блок 2 (div) — display: block — следующая строка
Блок 3 (div) — display: block — и ещё одна строка
display: inline — span-элементы текут в строку (без padding/border — те стакаются вертикально)

Текст строки: красный bold зелёный italic синий underline фиолетовый золотой bold — все на одной строке, перенос при нехватке места.

display: none — элемент полностью скрыт, место не занимает (только на блочных div)

Первый параграф — видимый.

ЭТОТ БЛОК СКРЫТ (display:none на div) — не занимает место

Второй параграф — видимый. Между параграфами нет пробела — скрытый div не занял место.

overflow: hidden — содержимое обрезается по границе блока
overflow:hidden — этот текст очень длинный и будет обрезан при выходе за границу блока фиксированной высоты
overflow:visible — текст выходит за пределы блока фиксированной высоты и виден снаружи блока
Единицы ширины: px, %, vw
width: 200px — фиксированная
width: 50% — половина родителя
width: 75vw — 75% ширины viewport (768px)
width: 100% — вся доступная ширина
Единицы высоты: px, vh — блочные div с фиксированной высотой
h:20px
h:30px
h:40px
h:8vh
← vh от высоты окна 720px · 8vh = ~58px
Наследование CSS — дочерние элементы наследуют color, font-size, font-style
Родитель: color=красный, font-size=13px. Дочерний span наследует цвет (bold локально).
Вложенный div: наследует красный + добавляет italic. Внук-span: цвет → синий, курсив унаследован.
Вложенные блоки — порядок отрисовки DOM (дочерние поверх родителя)
Уровень 1 — красный фон
Уровень 2 — синий
Уровень 3 — зелёный
Уровень 4 — жёлтый (контент)

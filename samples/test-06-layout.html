display: block — каждый блок начинается с новой строки, занимает всю ширину
Блок 1 (div) — display: block — вся ширина
Блок 2 (div) — display: block — следующая строка
Блок 3 (div) — display: block — и ещё одна строка
display: inline — элементы текут в строку как текст, ширина по содержимому

Текст строки: span inline 1 span inline 2 span inline 3 span inline 4 — все на одной строке, перенос при нехватке места.

display: none — элемент полностью скрыт, место не занимает

Первый абзац — видимый. ЭТОТ ТЕКСТ СКРЫТ (display:none) — не занимает место Третий абзац — видимый. Между ними нет пробела — скрытый элемент не занял место.

overflow: hidden — содержимое обрезается по границе блока
overflow:hidden — этот текст очень длинный и будет обрезан при выходе за границу блока фиксированной высоты
overflow:visible — текст выходит за пределы блока фиксированной высоты и виден снаружи
Единицы ширины: px, %, vw — блоки разной ширины
width: 200px — фиксированная
width: 50% — половина родителя
width: 75vw — 75% ширины viewport (768px)
width: 100% — вся доступная ширина
Единицы высоты: px, vh — блоки фиксированной высоты

h:20px h:30px h:40px h:8vh (~58px) ← vh зависит от высоты окна 720px

Наследование CSS — дочерние элементы наследуют color, font-size, font-style
Родитель: color=красный, font-size=13px. Дочерний span наследует цвет (bold добавлен локально).
Вложенный div: наследует красный, добавляет italic. Внук-span: переопределяет цвет на синий, курсив наследован.
Вложенные блоки — порядок отрисовки DOM (дочерние поверх родителя)
Уровень 1 — красный фон
Уровень 2 — синий
Уровень 3 — зелёный
Уровень 4 — жёлтый (контент)

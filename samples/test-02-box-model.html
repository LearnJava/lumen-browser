CSS Box Model: margin (голубой) → border (синий) → padding (зелёный) → content (жёлтый)
MARGIN — внешний отступ
BORDER 5px solid
PADDING — внутренний отступ
CONTENT — содержимое блока
Ширина рамки (border-width) — блочные элементы
border: 1px solid — тонкая рамка
border: 2px solid — стандартная
border: 4px solid — заметная
border: 7px solid — толстая
Цвет каждой стороны (border-top/right/bottom/left-color)
Верх — красный · Право — зелёный · Низ — синий · Лево — фиолетовый
Только отдельные стороны рамки
border-top: 3px solid red
border-left: 5px solid green — как цитата
border-bottom: 3px solid blue
Внутренний отступ (padding) — блочные элементы
padding: 2px 6px — минимальный
padding: 8px 16px — стандартный
padding: 16px 32px — увеличенный
box-sizing: content-box vs border-box (оба width: 400px)
content-box: width=400px + 2×6(pad) + 2×4(border) = 424px реальная
border-box: width=400px ровно — padding и border внутри

# BUG-219

**Статус:** FIXED 2026-07-04 (DEBTOR)
**Компонент:** image/paint
**Тест:** TEST-18 (KNOWN_DEBTOR 2.11%), TEST-19 (KNOWN_DEBTOR 9.05%)

## Описание

Тонкое расхождение пикселей при downscale растровых картинок относительно Edge.
После фикса геометрии (BUG-180, «image bottom gap») TEST-18 даёт 2.11% — diff
размазан тонким AA по всем фото; ни одна ячейка не разъезжается структурно
(порог `>60` всего 0.43% → геометрия/позиция верны).

## Расследование (P3, 2026-07-04)

Гипотеза бага: перейти на «более близкий к браузеру kernel» (Lanczos/bicubic)
и снять остаток. Гипотеза **проверена эмпирически и отклонена**.

Реализованы и прогнаны два альтернативных разделимых фильтра вместо
производственного box-фильтра (`resize_area_avg`, area-averaging):
- **Lanczos-3** (sinc·sinc, support 3, негативные лепестки);
- **Mitchell-Netravali** (bicubic B=C=1/3, support 2 — фильтр Skia/Blink
  medium-quality).

Замеры одной сессии (один и тот же gdigrab-пайплайн, сравнение относительное —
абсолютные числа сессии смещены от канонического CI-эталона: box в этой среде
даёт TEST-18 6.12% против коммит-baseline 2.11%, т.е. среда завышает diff):

| Kernel   | TEST-18 | TEST-19 |
|----------|---------|---------|
| box (area-avg, prod) | 6.12% | 8.97% |
| Mitchell bicubic     | 5.34% | 9.55% |
| Lanczos-3            | 7.55% | 9.16% |

**Ни один kernel не выигрывает равномерно:** Mitchell чуть улучшает TEST-18
(−0.78 п.п.), но регрессирует TEST-19 (+0.58 п.п.); Lanczos хуже box на обоих.
Более резкие ядра выменивают выигрыш на резкости контента одного теста на потерю
на другом (ringing/гало на высококонтрастных фото). Box-фильтр — сбалансированный
центр и совпадает с философией зафиксированного baseline.

## Вывод

Остаток — **inherent** resampling + AA-парность против downscale-фильтра Edge
(проприетарный, контент-адаптивный Skia-пайплайн: mipmap/box/lanczos по
коэффициенту), а не геометрический/логический дефект движка. Класс rule 2/3
(edge-AA / kernel-parity), аналог BUG-247/BUG-176/BUG-184.

Производственный код оставлен без изменений (`resize_area_avg` во всех
downscale-путях: femtovg `resolve_image_for_rect`, `cpu_raster`, wgpu `renderer`).
Альтернативные ядра не приняты — регрессируют один из двух baseline и не дают
чистого выигрыша. TEST-18/19 остаются KNOWN_DEBTORS с прежними baseline
(2.11% / 9.05%); цель <0.5% недостижима простым сменом kernel.

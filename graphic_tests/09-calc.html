LUMEN — CSS MATH (calc / min / max / clamp + trig + log)
calc(...) — арифметика над длинами
width: 100px
эталон 100px
calc(100px + 50px)
= 150
calc(200px - 30px)
= 170
calc(50px * 4)
= 200
calc(900px / 3)
= 300
calc((100px + 50px) * 2)
= 300
calc(20em + 10px) (em = 13px)
= 270
min(...) / max(...) — выбор аргумента
min(150px, 300px)
= 150
min(400px, 250px, 150px)
= 150
max(150px, 300px)
= 300
max(50px, 100px, 250px, 80px)
= 250
clamp(min, val, max) — зажим
clamp(100px, 200px, 300px) → val в диапазоне
= 200
clamp(100px, 50px, 300px) → val ниже min
= 100
clamp(100px, 400px, 300px) → val выше max
= 300
clamp(300px, 100px, 200px) → min > max → побеждает min
= 300
trig — sin/cos/sqrt/abs/pow
calc(sin(0.5235988) * 200px) ≈ sin(30°)·200
≈ 100
calc(cos(0) * 200px) ≈ 200
= 200
calc(sqrt(10000) * 1px)
= 100
calc(pow(2, 7) * 1px)
= 128
calc(hypot(150px, 200px))
= 250
calc(abs(-180px))
= 180
вложенные комбинации
min(calc(50px + 100px), max(80px, 200px))
= 150
clamp(60px, calc(40px * 3), 200px)
= 120

min-width / max-width — clamp фактической ширины
width: 80px (~ маленький)
width: 800px (~ большой)
width: 80px; min-width: 200px → реально 200px
width: 800px; max-width: 300px → реально 300px
width: 50; min-width: 150; max-width: 100 → min побеждает = 150

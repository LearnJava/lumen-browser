Тестовые файлы для графического тестирования находятся в папке `graphic_tests/` (в корне репо), а не в `samples/`.

Тест считается пройденным при расхождении ≤ 1% (порог pixel diff).

**Why:** Пользователь явно указал — не путать с `samples/`; порог 1% — официальный критерий pass/fail.
**How to apply:** При поиске тестовых страниц — смотреть в `graphic_tests/`. При оценке результатов `run.py` — PASS ≤ 1%, FAIL > 1%.

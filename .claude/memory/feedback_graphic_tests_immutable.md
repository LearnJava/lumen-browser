Тест-страницы в `graphic_tests/` — неприкосновенный эталон. При провале теста:

- **Запрещено:** удалять тест, упрощать HTML, менять CSS, снижать порог — любое изменение тест-файла.
- **Разрешено:** исправлять код браузера (движок, парсер, layout, paint).
- **Разрешено:** создавать новые тест-файлы, если нужен анализ или изоляция бага.
- **Порог расхождения с эталонным браузером:** 0.5% пикселей. Тест считается пройденным если diff ≤ 0.5%.

**Why:** тест-страницы представляют корректный CSS по версии Edge. Переписать тест чтобы он прошёл — это ложный positive: движок не улучшился, планка была снижена.

**How to apply:** если тест падает — ищи и исправляй баг в коде браузера, не трогая HTML/CSS тест-файла.

При реализации любого нового CSS-свойства в lumen-layout или lumen-paint обязательно:

1. Добавить объект(ы) в соответствующий тест из серии `graphic_tests/01-19` (или создать новый файл если свойство не покрыто).
2. Добавить свойство в `graphic_tests/1000000-final.html` — финальный тест, где все реализованные свойства видны в одном окне.
3. Обновить таблицу в `graphic_tests/COVERAGE.md` — добавить строку для нового свойства, указать файлы покрытия.

**Why:** пользователь явно зафиксировал этот протокол.  
**How to apply:** срабатывает при любой задаче, связанной с реализацией CSS property (commit в lumen-layout/lumen-paint затрагивает apply_declaration или ComputedStyle). Не ждать отдельной просьбы обновить тесты — делать в том же коммите.

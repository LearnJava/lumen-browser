# P2-wpt metadata root

Passed to `wptrunner` as `--metadata` (`tests/wpt/run_smoke.py`). Holds:

- `MANIFEST.json` — regenerated by `wptrunner --manifest-update` on every run; gitignored, not committed.
- `.cache/` — wptrunner's own manifest mtime cache; gitignored, not committed.
- `.ini` per-test expectation files — introduced in S5 ("Expectations + curated subset",
  `docs/tasks/p2-wpt-integration.md`); committed once they exist.

This file exists so the otherwise-empty directory is tracked before S5 adds real content.
